The Slider on the homepage

HTML code is located in the file 'index.html'. On the path 'js/main.js' is initialization of the slider.

Slides are displayed by the layers with an extensive range of animation and controlled by the attributes of HTML tags.
Pictures and text blocks can act by the slide layers. Main options of the layers:

  • data-position - position of the element inside the slider. Format: "top,left" in pixels
  • data-in / data-out - type of the in-animation/out-animation (default is left)
    possible values: fade, none ,left, topLeft, bottomLeft, right, topRight, bottomRight, top, bottom
    none lets the element just appear/disappear at the desired location
    the direction-type values define the direction in which the element is animated.
    example:  data-in="left":the element will start right side of the slider and move left till its at the target location
    example: data-out="left": the element will move left till its out of the slider
  • data-ease-in / data-ease-out - easing for the animations (you can use all in jquery-ui contained easing methods)
  • data-fixed - the Element won´t be animated and will be visible from the beginning of the slide (no value is needed)
  • data-step - You can group your elements in different steps. All animation of one step will start at the same time
    (maybe with your choosen element-specific data-delay). Elements of the next step will not start before the previous step is finished.
    If an element has no data-step attribute the plugin will thread it as having data-step=”0″.
    example: 
    <p data-position="10,50" data-step="0">I start first</p>
<p data-position="10,50" data-step="0">me to</p>
<a data-position="10,50" data-step="1">I start when both paragraphs are finished</a>
  • data-delay - time in ms before the in transition starts (in the current step / see steps)
  • data-time - time after which the elements animation is complete. It will start at the beginning of the slide/step, or after its specificed delay
  • data-special - The only value currently available for data-special is cycle.
    cycle will trigger the out transition for the previous element in the current step.
    (remember to use data-delay for elements with cycle, or they will start at the same time the element to cycle starts – that will break the whole fun)
    example:
    <p data-position="10,50" data-step="1">I will transition out when the second p transition starts</p>
<p data-position="10,50" data-step="1" data-special="cycle" data-delay="1000">I will transition out when the third p transition starts</p>
<p data-position="10,50" data-step="1" data-special="cycle" data-delay="2000">I m the third p</p>

You can change the slide background by replacing 'data-bg' attribute:

<div class="slide" data-bg="#f6b88c">

The Gallery in the page 'About Us' (about.html)

For the adding a new image you will need to copy a block of code between the tags <div class="about-i"></div> and replace reference to the picture and preview.

In the attribute 'scr' of <img> tag will need indicate the path to preview.
You can add a preview of image as the same ('grid' display style), and of arbitrary size ('masonry' display style).

<div class="about-i">
	<a class="modal-img" data-fancybox-group="about" href="img/about/preview1.jpg">
		<img src="img/about/preview1.jpg" alt="">
	</a>
</div>

For a full-size image the path specified in the attribute 'href' of tag <a>.
For a video you will need to add a class 'about-video' to the <div class="about-i"> element in additionally.
Copy the URL of youtube video and add in a line '&fs=1&autoplay=1' to display it on the site.

<div class="about-i about-video">
	<a class="modal-img" data-fancybox-group="about" href="https://www.youtube.com/watch?v=6D-A6CL3Pv8&fs=1&autoplay=1">
		<img src="img/about/preview2.jpg" alt="">
	</a>
</div>

Contact page

Change a coordinates on the map

To change the locate on the map, you will need to determine the coordinates in Google Maps by clicking at a certain point (as shown in the image below),

and then add the coordinates in the file 'contacts.html'.

function initialize() {
	var mapOptions = {
		zoom: 15,
		scrollwheel: false,
		center: new google.maps.LatLng(40.440128, -79.974326)
	};
	var map = new google.maps.Map(document.getElementById('contacts-map'),
		mapOptions);
	var marker = new google.maps.Marker({
		position: map.getCenter(),
		icon: 'img/marker.png',
		map: map
	});
}
google.maps.event.addDomListener(window, 'load', initialize);
